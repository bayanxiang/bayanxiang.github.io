Android Studio 是谷歌推出的一个Android集成开发工具，基于IntelliJ IDEA. 类似 Eclipse ADT，Android Studio 提供了集成的 Android 开发工具用于开发和调试。

Android Studio

Android Studio 是开发 Android 应用程序的官方 IDE，基于 Intellij IDEA。

  • 下载

进入Andriod Studio官网点击DOWNLOAD ANDROID STUDIO下载。

  • 安装

点击已下载的安装包，逐步进行安装，图示如下：

①点击Next

②点击Next

③安装路径建议安装在除C盘以外的地方，更改后点击Next

④点击Install

⑤点击Next

****

⑥Start Android Studio根据自己需求选择是否启动，然后点击Finish

⑦第一次安装android studio时候弹出unable to access android sdk add-on list原因是你电脑没有SDK而且你下载的android studio又是不带SDK的，点击Cencel即可进入下一环节

⑧此处提示找不到Android SDK，点击Next

⑨更改位置，然后点击Next

⑩点击Finish

⑪点击Finish完成安装

⑫Welcome to Android Studio

  • 使用

①点击Create New Project创建第一个项目

②对相关信息进行配置，点击Next

③项目启动后主界面

④Run

踩坑指南

①Build时报错：你的主机中的软件中止了一个已建立的连接。

错因：笔记本打开了移动热点

解决办法：关闭移动热点，重新clean

②Session 'app': Installation did not succeed.The application could not be installed: INSTALL FAILED INSUFFICIENT STORAGE Retry

错因：Session 'app':安装失败。无法安装应用程序:安装失败，存储空间不足，请重试

解决办法：点击最上面一行菜单栏的Build选项，选择 Clean Project，清理缓存之后，重启Android Studio就好了
